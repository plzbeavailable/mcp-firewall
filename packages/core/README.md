# @ziwansi/mcp-firewall-core

Core proxy engine and security pipeline for MCP Firewall.

This is an internal package. Most users should install [`@ziwansi/mcp-firewall`](https://www.npmjs.com/package/@ziwansi/mcp-firewall) instead.

```bash
npm install -g @ziwansi/mcp-firewall
```

## Architecture

- **Proxy Engine** — stdio, HTTP, and SSE transport support
- **Security Pipeline** — 6-layer defense-in-depth
- **Observability** — metrics, audit logging, token tracking, health checks

See [GitHub](https://github.com/plzbeavailable/mcp-firewall) for full documentation.
