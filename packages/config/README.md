# @ziwansi/mcp-firewall-config

Configuration system for MCP Firewall — Zod schemas, YAML loading, hot-reload support.

This is an internal package. Most users should install [`@ziwansi/mcp-firewall`](https://www.npmjs.com/package/@ziwansi/mcp-firewall) instead.

```bash
npm install -g @ziwansi/mcp-firewall
```

## Features

- **Zod Schemas** — Full type-safe config validation
- **YAML/JSON** — Both formats supported
- **Hot Reload** — Watch config file for changes via chokidar
- **Presets** — `readonly` and `full` built-in config templates

See [GitHub](https://github.com/plzbeavailable/mcp-firewall) for full documentation.
